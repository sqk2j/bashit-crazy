1. Elements & Structure

a) HTML = HyperText Markup Language

  • used to give content to web pages
  • used to instruct web browser on how to structure content

b) Element Content

  • content of HTML element is the information between opening and closing tags
  • <p> = opening tag
  • Hello World! = content
  • </p> = closing tag
<p>Hello World!</p>

c) HTML Tag

  • Syntax for single HTML tag = opening angle bracket "<" followed by element name and closing angle bracket ">"
  • ex. opening <div> tag
<div>

d) Element Content

  • content of HTML element = information between the opening and closing tags
<h1>I'm the h1 element CONTENT!</h1>

e) Closing Tag

  • used to denote the end of an HTML element
  • syntax for closing tag is left angle brack "<" followed by a forward slash "/" then the element name and a right angle bracket to close ">"
<body>...</body>
HTML ELEMENTS
<body> Represents content of HTML document. Content inside <body> tags rendered on web browsers
Note: CAN ONLY BE ONE <body> element in a document 		</body>
<h1>-<h6> HTML can use six different levels of heading elements. Ordered from hightest level <h1> to lowest <h6> </h1>-</h6>
<div> Used as a container that divides HTML document into sections
<div> elements can contain flow content like headings, paragraphs, links, images, etc. 		</div>
