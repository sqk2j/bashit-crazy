1. Intro

The command line is a text interface for your computer.
It is a program that takes in commands, passing them along to the computer's operating system to run.
You can run programs, scripts to automate tasks, and combine simple commands to handle difficult tasks.

2. Navigating Filesystem: Common Commands

  • "ls": list contents of a directory
  • "pwd": print working directory
  • "cd": change directory
  • "cd .." : move up one directory
  • "mkdir": create a mew directory
  • "touch": creates a new file inside working directory

3. Options

Options modify the behavior of commands.

ex. "ls -a"

  • "ls": lists all files and directories in working directory
  • "-a": modifies behavior of "ls" to also list hidden files

a) Other "ls" Options

  • "-a": lists all contents, including hidden files & directories
  • "-l": lists all contents of a directory in long format
  • "-t": order files and directories by the time they were last modified
  • "ls -alt": lists all contents, including hidden files and directories, in long format, ordered by date & time they were last modified
